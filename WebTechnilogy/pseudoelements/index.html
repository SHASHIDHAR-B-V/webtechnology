Pseudo Element is used to target particular part of an Element

  1. First Letter
  2. First Line
  3. After
  4. Before
  5. Marker
  6. Selection

 syntax for pseudo element 
 selector::pseudo-element
     {
      property : value;
     }

Pseudo Element - First Letter selects only first letter of an Element h1::first-letter

Pseudo Element First line targets only first line of an Element h1::firstline
